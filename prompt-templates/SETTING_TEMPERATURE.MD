# Setting Temperature

Some AI tools let you set a "temperature" rating (0.1 to 1.0):

## Temperature Ranges

### Lower Temperature (0.1-0.4)
More predictable, factual, conservative

**Use for:**
- Data analysis
- Technical documentation
- Factual summaries

### Higher Temperature (0.7-1.0)
More creative, varied, exploratory

**Use for:**
- Brainstorming
- Creative content
- "Opportunities to delight"
